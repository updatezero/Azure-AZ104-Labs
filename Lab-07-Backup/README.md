# Lab 07 – Azure Recovery Services

## Objective

The goal of this lab was to explore Azure Recovery Services and understand how Microsoft Azure protects virtual machines, backup data, and disaster recovery resources.

During this lab I explored the Recovery Services Vault, backup configuration, backup policies, monitoring features, alerts, metrics and diagnostics.

---

## Topics Covered

- Azure Recovery Services Vault
- Backup Dashboard
- Backup Items
- Backup Policies
- Backup Jobs
- Backup Alerts
- Azure Monitor Metrics
- Diagnostic Settings
- Advisor Recommendations

During this lab I learned:

- What an Azure Recovery Services Vault is
- Why backups are critical for business continuity
- Difference between Backup and Site Recovery
- How Azure stores protected workloads

- How Diagnostic Settings integrate with Azure Monitor
