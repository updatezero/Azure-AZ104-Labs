# Lab 05 – Azure Virtual Machines

## Objective

This lab focuses on deploying, configuring, and exploring Azure Virtual Machines. It covers VM sizing, storage, networking, availability, operating system settings, security, and supporting Azure infrastructure components.

---

## Technologies Used

- Microsoft Azure
- Azure Virtual Machines
- Managed Disks
- Virtual Network (VNet)
- Network Interface (NIC)
- Public IP
- Network Security Group (NSG)
- Azure Availability Zones
- Application Security Groups (ASG)
- Azure Network Manager

---

## Tasks Completed

- Reviewed the virtual machine overview.
- Explored VM sizing options.
- Reviewed Availability Zones and Availability Sets.
- Examined managed OS disks.
- Reviewed VM extensions.
- Explored operating system settings.
- Reviewed VM configuration.
- Examined networking configuration.
- Reviewed Network Security Groups.
- Explored Load Balancer integration.
- Reviewed Application Security Groups.
- Explored Azure Network Manager.
