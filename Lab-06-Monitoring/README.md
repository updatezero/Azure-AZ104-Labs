# Lab 06 – Azure Monitoring

## Overview

This lab focuses on Azure Monitor, Microsoft's centralized monitoring platform used to collect, analyse and visualize telemetry across Azure resources.

The lab introduces monitoring concepts such as metrics, logs, alerts, service health and infrastructure monitoring.

---

## Objectives

- Explore Azure Monitor
- Review Activity Logs
- Configure Metrics
- Analyse CPU utilization
- Explore Azure Monitor Logs
- Review Azure Workbooks
- Understand Azure Service Health
- Explore Change Analysis
- Review Virtual Machine Insights
- Explore Azure Managed Prometheus

## Key Learnings

- Azure Monitor centralizes monitoring across Azure.
- Metrics provide performance information.
- Logs provide operational information.
- Alerts automate incident detection.
- Workbooks improve operational visibility.
- Service Health monitors Azure platform availability.
- VM Insights provide deep infrastructure monitoring.
- Change Analysis helps identify infrastructure modifications.
- Monitoring is a critical responsibility of Azure Administrators.
