# Lab 04 - RBAC

## Topics Covered

- Azure Role-Based Access Control (RBAC)
- Identity and Access Management (IAM)
- Built-in Azure roles
- Owner role
- Virtual Machine Contributor role
- Role assignments
- Subscription scope
- Resource Group scope
- Permission inheritance
- Principle of Least Privilege

## Key Learnings

- Azure RBAC controls access to Azure resources.
- Permissions can be assigned at different scopes.
- Subscription permissions are inherited by Resource Groups and resources.
- "This resource" indicates a role assigned directly at the current scope.
- "Inherited" indicates a role received from a parent scope.
- Virtual Machine Contributor allows managing virtual machines without full administrative permissions.
