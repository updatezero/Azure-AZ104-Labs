# Lab 04 - Azure RBAC

## Objective

Learn how Azure Role-Based Access Control (RBAC) works by assigning built-in roles, understanding permission scopes, and verifying inherited permissions.

## Tasks Completed

- Reviewed existing role assignments
- Verified Owner permissions on the subscription
- Created a new Virtual Machine Contributor role assignment
- Assigned the role to my Azure account
- Compared Subscription and Resource Group scopes
- Verified inherited permissions

## Azure Services Used

- Azure Subscription
- Resource Group
- Azure RBAC
- IAM (Identity and Access Management)

## Skills Practiced

- RBAC
- Built-in roles
- Role assignments
- Permission inheritance
- Least privilege
- Azure IAM
